# news-manga — AIニュース4コマ生成

ニューストピックを1つ受け取り、(1) チビキャラ4コマ風マンガ画像を gpt-image-2 で
1枚生成し、(2) Threads向けの投稿文を作る。

## 入力
- ニューストピック（例: 「Fable5が米国政府指令で使用停止」）
- トーン指定があれば従う（無ければ「自分ごと・共感系のゆるい実況」）

## 固定スタイル（毎回統一・崩さない）
- チビキャラ = 黒髪・グレーパーカーの日本人少年、大きい目
- 背景 = カフェ/デスク（ノートPC・コーヒー・観葉植物）
- 構図 = 縦長2x2の4コマ、白い枠線、明るいパステル
- 上部に大きい黒の日本語タイトル（2行）
- セリフは各コマ短く（画像モデルは長文・小さい文字が苦手）

## 手順

### 1. 4コマ構成を考える
トピックを「起→驚→困→オチ」の4コマに落とす。各コマに短いセリフ/吹き出しを付ける。
タイトルは2行のキャッチー日本語。

### 2. プロンプトファイルを書く
以下の形式で一時ファイルに書く（英語の構造指示＋日本語セリフ混在）。
スタイル指示はスクリプト側で自動付与されるので、ここでは**タイトルと4コマ内容だけ**書く:

```
At the very top, large bold black rounded Japanese title text on two lines:
「<タイトル上段>」「<タイトル下段>」
Panel ① <情景の英語説明>. Bubble:「<日本語セリフ>」
Panel ② <情景>. Bubble:「<セリフ>」
Panel ③ <情景>.
Panel ④ <情景>. Thought:「<セリフ>」
```

Write ツールで `C:\Users\User\.claude\scripts\_manga_prompt.txt` に保存する。

### 3. 画像生成（課金あり: gpt-image-2 を1枚呼ぶ）
```
py "C:\Users\User\.claude\scripts\gen_news_manga.py" --prompt-file "C:\Users\User\.claude\scripts\_manga_prompt.txt" --slug <英小文字スラッグ>
```
出力先: `GPT-auto\ニュース画像\<slug>_<日時>.png`

### 4. 画像を Read して確認
日本語の崩れ・レイアウトを目視。大きく崩れていたら 4コマ構成のセリフを短くして再生成。

### 5. Threads投稿文を作る
以下のルールで作成しチャットに提示（コードブロックで）:
- 冒頭フックで「自分ごと」化（呼びかけ＋絵文字）
- 短い改行多めでスマホで読みやすく
- 感情の絵文字で共感誘導
- 末尾に質問でコメント/リプを促す
- ハッシュタグは4個に厳選（多すぎ厳禁）

### 6. 投稿先の確認
投稿はユーザーが手動で行うことが多い。自動投稿を求められた場合のみ
`sns_post.py` を使い、投稿先アカウントを必ず確認してから実行する。

## 注意
- 画像生成は OpenAI 課金が発生する（1枚）。トピックが固まってから実行する。
- 既定の出力先フォルダが無ければスクリプトが自動作成する。
